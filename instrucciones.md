# Sistema de Autenticación y Panel de Perfil - CEPLAN

Una aplicación web full-stack interactiva desarrollada con **ASP.NET Core Blazor (.NET 10)** y **SQL Server**.

## 🚀 Cómo ejecutar el proyecto de forma local
1. Clona este repositorio en tu máquina local.
2. Abre **SQL Server Management Studio (SSMS)** y ejecuta por completo el contenido del archivo `schema.sql` incluido en la raíz para generar la base de datos, las tablas y los usuarios de prueba.
3. Modifica la cadena de conexión `DefaultConnection` en el archivo `appsettings.json` para que apunte al nombre de tu instancia local de SQL Server.
4. Ejecuta la aplicación desde Visual Studio presionando `Ctrl + F5`.
